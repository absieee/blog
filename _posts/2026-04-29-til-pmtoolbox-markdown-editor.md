TIL there exists something called [PM Toolbox](https://pmtoolbox.app/)

I was using the out of the box MD editor in Cursor so far and there had been a number of occassions where I got so fed up with the experience, I defaulted to relying on LLM and auto-correct to update the smallest of items in an MD file.

With the PM Toolbox editor that I accidentally discovered just now while trying to find something else, I can now easily write MD files in the way I got used to writing them in [Obsidian](https://obsidian.md/) a couple months ago but with the added bonus of it being even faster because now I can use in-line slash commands to add headings, images, links, diagrams without faffing around with markdown syntax (which you still have to do a little bit of in Obsidian).

I have tried in the past to switch to [Nimbalyst](https://nimbalyst.com/), other markdown editors like [this](https://marketplace.cursorapi.com/items/?itemName=zaaack.markdown-editor), or just linking everything up to Obsidian directly but this extension really is beating them all - atleast for now.

Let's see if it holds up.

P.S. I wrote this post completely using the extension to test it out and the experience has been friction-less! 

P.P.S. I don't have to waste time on creating or invoking md template skills now at the precious cost of tokens because this extension allows me to create templates out-of-the-box that I can reuse very easily.
